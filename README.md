# AROYALPRIESTHOOD

A modern, mobile-first Christian baby announcement website for the arrival of a baby boy.

## Scripts

| Command              | Description                      |
| -------------------- | -------------------------------- |
| `npm run dev`        | Start Next.js dev server         |
| `npm run build`      | Production build                 |
| `npm start`          | Start production server          |
| `npm run lint`       | Run ESLint                       |
| `npm run typecheck`  | Run TypeScript type checking     |
| `npm run format`     | Format code with Prettier        |
| `npm run format:check` | Check formatting              |
| `npm test`           | Run Playwright tests             |
| `npm run test:ui`    | Run Playwright tests with UI     |

## Tech Stack

- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **Styling**: Plain CSS with CSS custom properties
- **Fonts**: Cinzel, Montserrat, Cormorant Garamond, Playfair Display, Inter (via `next/font`)
- **Icons**: lucide-react

## Features

- Responsive navigation with mobile hamburger menu
- YouTube video embeds
- Testimony timeline
- Animated baby name reveal with confetti
- Gift registry with mailto links
- Scroll-triggered reveal animations
- Social sharing metadata (Open Graph + Twitter)
- Accessibility (aria attributes, semantic HTML, reduced motion)
